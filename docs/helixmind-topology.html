5 Spiral Memory Levels + Web Knowledge

Hierarchische Wissens-Spirale mit Evolution & Decay. Definiert in src/types.ts:17 als SpiralLevel = 1 | 2 | 3 | 4 | 5. Web Knowledge wird als virtuelles L6 gezaehlt, lebt aber als getaggte L2-Nodes.

L1
Focus
Hoechste Relevanz. Aktiver Arbeitskontext. Voller Inhalt wird in System-Prompt injiziert. Semantik 45%, Recency 30%, Connection 15%, Type 10%.
Score ≥ 0.7
L2
Active
Moderate Relevanz. Aktueller Projektkontext. Summary oder voller Inhalt. Web Knowledge Nodes landen hier (Tag: web_knowledge).
Score ≥ 0.5
L3
Reference
Niedrigere Relevanz. Hintergrundwissen. Content summarized, erste 200 Zeichen in Context.
Score ≥ 0.3
L4
Archive
Minimale Relevanz. Stark komprimiert, erste 100 Zeichen. Historisches Wissen.
Score ≥ 0.1
L5
Deep Archive
Tiefste Ebene. Nur Metadata + Kurz-Summary, erste 50 Zeichen. Wird nie geloescht, bleibt abrufbar.
Score < 0.1
~L6
Web Knowledge
Virtuelles Level. DuckDuckGo-Suche + HTML-Extraktion im Hintergrund. Gespeichert als L2-Nodes mit web_knowledge Tag. 0 LLM Tokens, rein regelbasiert.
virtuell

Komponenten-Uebersicht

Agent Loop

src/cli/agent/loop.ts

Herzstueck: LLM → Tool → Result → Repeat. Max 200 Iterationen. Trimming ab 85% Context. AbortController.

Provider Layer

src/cli/providers/

8 Provider, 2 Klassen. AnthropicProvider (native) + OpenAIProvider (native + 6x baseURL-Routing). Rate-Limiter mit Exponential Backoff.

Spiral Memory Engine

src/spiral/engine.ts

5 Levels (Focus → Deep Archive). SQLite + sqlite-vec. HuggingFace Embeddings (384d). Evolution & Decay. Nie loeschen.

Web Knowledge Enricher

src/spiral/cloud/

Hintergrund: Topic Detection → DuckDuckGo → HTML Parse → Spiral L2. 0 LLM Tokens. 10min Cooldown.

Context Assembler

src/cli/context/assembler.ts

10 bedingte Sektionen. Trimmer haelt Konversation unter Context-Limit. Session Buffer fuer Short-Term.

Validation Matrix

src/cli/validation/

4 Phasen: Static (15+ Checks) → Dynamic (LLM) → Spiral → Autofix (3x). 9 Module. Eigener Fast-Model.

Brain Visualization

src/cli/brain/

HTTP + WebSocket Server (:9420-9440). 3D Helix (Three.js). Live Events. Findings Panel. Relay zum Web-Dashboard.

Session Manager

src/cli/sessions/

Multi-Session: /security und /auto im Hintergrund. Tab-Bar. Eigene History + Controller + Undo pro Session.

Jarvis Daemon

src/cli/jarvis/

Autonomer Agent: Scheduler, Triggers, Orchestrator, Skill-Builder, Learning. Notifications: Telegram, Email, Webhooks, ElevenLabs Voice.

Browser Control

src/cli/browser/

Puppeteer-Core + Chrome-Finder. 6 Tools: open, navigate, screenshot, click, type, close. VisionProcessor fuer Analyse.

Plan & Swarm Engine

src/cli/agent/plan-*.ts, swarm.ts

Plan Lifecycle + Step Execution. Swarm: Task Decomposition → Parallel Workers mit File Lock.

Checkpoint System

src/cli/checkpoints/

Auto-Snapshots bei jedem Tool-Call. ESC ESC → Browse & Revert. Chat + Code Revert. Memory-Management.
