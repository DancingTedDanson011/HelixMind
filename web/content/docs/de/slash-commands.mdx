# Slash-Befehle

Tippe waehrend einer interaktiven Chat-Session `/` gefolgt von einem Befehlsnamen. Diese Befehle steuern HelixMinds Verhalten, verwalten Memory und loesen spezielle Modi aus.

> Einige Befehle erfordern einen kostenlosen Login. Diese sind im Hilfe-Menue mit einem Schloss-Symbol markiert und im Open Source Modus aus der Autovervollstaendigung ausgeblendet. Fuehre `helixmind login` aus um alle Befehle freizuschalten.

## Chat & Interaktion

| Befehl | Beschreibung |
|--------|-------------|
| `/clear` | Gesprächsverlauf löschen (Spiral-Memory bleibt erhalten) |
| `/model` | Interaktiver Modell-Picker zum Wechseln des LLM-Modells |
| `/keys` | API-Keys interaktiv verwalten |
| `/help` | Alle verfügbaren Slash-Befehle anzeigen |
| `/exit` / `/quit` | HelixMind-Session beenden |

## Spiral-Memory

| Befehl | Beschreibung |
|--------|-------------|
| `/spiral` | Schnelle Spiral-Status-Übersicht |
| `/feed` | Dateien interaktiv in die Spiral einspeisen |
| `/context` | Kontext-Assembly für das aktuelle Gespräch anzeigen |
| `/compact` | Spiral-Evolution auslösen |
| `/tokens` | Token-Verbrauch pro Level anzeigen |

## Visualisierung & Brain

| Befehl | Beschreibung |
|--------|-------------|
| `/brain` | 3D-Brain-Visualisierung im Browser öffnen |
| `/brain local` | Brain auf projekt-lokalen Scope umschalten |
| `/brain global` | Brain auf globalen Scope umschalten |
| `/helix` | HelixMind Command Center öffnen |

## Berechtigungs-Modi

| Befehl | Beschreibung |
|--------|-------------|
| `/yolo` | YOLO-Modus umschalten (alle Operationen automatisch genehmigt) |
| `/skip-permissions` | Skip-Permissions-Modus umschalten |

## Konto & Auth

| Befehl | Beschreibung |

| `/login` | Browser-basierten Login-Flow starten |

## Tastenkürzel

| Kürzel | Aktion |
|--------|--------|
| **ESC ESC** (doppelt) | Notfall-Stopp — bricht alle Sessions ab + öffnet Checkpoint-Browser |
| **Ctrl+PageUp** | Zum vorherigen Session-Tab wechseln |
| **Ctrl+PageDown** | Zum nächsten Session-Tab wechseln |
| **Tab** | Befehl-/Pfad-Autovervollständigung |
| **Ctrl+C** | Aktuelle Eingabe löschen oder beenden |
