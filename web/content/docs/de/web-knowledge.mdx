# Web Knowledge Enricher

Der Web Knowledge Enricher **sucht automatisch im Internet** nach relevanter Dokumentation, während du arbeitest.

## Ablauf

1. **Topic Detection**: Erkennt Technologie-, Pattern- oder Error-Topics in deiner Nachricht
2. **Web-Suche**: DuckDuckGo-Suche nach relevanten Ergebnissen
3. **Wissensextraktion**: Code-Beispiele, Key-Points, Zusammenfassungen aus HTML
4. **Spiral-Speicherung**: Als L6 Web Knowledge Knoten mit Qualitäts-Score
5. **Live Brain-Update**: Popup-Notification + Partikel-Animation

## Explizite Web-Recherche

```
> Suche im Web nach Next.js 15 Server Actions Migrationsanleitung
```

Die AI nutzt das `web_research` Tool. sucht, fetcht, extrahiert und speichert die Ergebnisse.

## Was gespeichert wird

| Feld | Beschreibung |
|------|-------------|
| `content` | Extrahiertes Wissen |
| `source` | Original-URL |
| `quality` | Qualitäts-Score (0.0 - 1.0) |
| `topic` | Erkanntes Topic |
| `embedding` | 384d Vektor für semantische Suche |

## Datenschutz

- Alle Suchen über **DuckDuckGo** (kein Tracking)
- Geholter Inhalt wird lokal verarbeitet
- Nur extrahiertes Wissen wird gespeichert
- Für vollständig offline: [Ollama](/docs/ollama-offline). Web-Enrichment wird deaktiviert
