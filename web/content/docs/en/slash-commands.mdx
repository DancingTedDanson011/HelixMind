# Slash Commands

Type `/` followed by a command name during an interactive chat session.

> Some commands require a free login. These are marked with a lock icon in the help menu and hidden from autocomplete in Open Source mode. Run `helixmind login` to unlock all commands.

## Chat & Interaction

| Command | Description |
|---------|-------------|
| `/clear` | Clear conversation history (spiral memory stays intact) |
| `/model` | Interactive model picker — switch LLM mid-session |
| `/keys` | Manage API keys interactively |
| `/help` | Show all available commands |
| `/exit` / `/quit` | Exit HelixMind gracefully |

## Spiral Memory

| Command | Description |
|---------|-------------|
| `/spiral` | Quick status: node counts per level, edges, storage size |
| `/feed` | Feed files into spiral memory (file picker) |
| `/context` | Show how context is assembled (nodes per level, token usage) |
| `/compact` | Trigger evolution: compress, promote, decay nodes |
| `/tokens` | Token usage statistics across all levels |

## Visualization & Brain

| Command | Description |
|---------|-------------|
| `/brain` | Open 3D Brain Visualization in browser |
| `/brain local` | Switch brain to project-local scope |
| `/brain global` | Switch brain to global scope (all projects) |
| `/helix` | Open HelixMind Command Center (menu-driven interface) |

## Permission Modes

| Command | Description |
|---------|-------------|
| `/yolo` | Toggle YOLO mode — auto-approve ALL operations (⚠️ dangerous) |
| `/skip-permissions` | Toggle skip-permissions — auto-approve safe + ask, still prompt dangerous |

| Command | Description | Requires |

| `/monitor` | Start security monitor — passive/defensive/active | Free+ |
| `/sessions` | List all sessions/tabs with status | — |

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| **ESC ESC** (double) | Emergency stop — aborts all sessions + opens checkpoint browser |
| **Ctrl+PageUp** | Switch to previous session tab |
| **Ctrl+PageDown** | Switch to next session tab |
| **Tab** | Command/path autocomplete |
| **Ctrl+C** | Clear current input or exit |
