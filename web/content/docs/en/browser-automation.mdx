# Browser Automation & Vision

HelixMind can **control a real Chrome browser** via Puppeteer â€” navigate, click, type, take screenshots, and analyze visual content.

## Why Browser Automation?

| Use Case | Example |
|:---------|:--------|
| **Web scraping** | Extract data from dynamic websites |
| **Visual testing** | Verify UI rendering, compare screenshots |
| **Workflow automation** | Log in, fill forms, click buttons |
| **Documentation** | Capture screenshots for tutorials |
| **Debugging** | Inspect live web app state |

## How It Works

1. **Chrome Detection** â€” Finds locally installed Chrome/Chromium
2. **Puppeteer Launch** â€” Starts headless or headed browser
3. **Tool Execution** â€” Navigate, click, type, screenshot
4. **Visual Analysis** â€” LLM analyzes screenshots (if vision model available)
5. **Cleanup** â€” Browser closes automatically

## Available Tools

| Tool | Description |
|:-----|:------------|
| `browser_open` | Open Chrome browser (headless/headed) |
| `browser_navigate` | Navigate to URL |
| `browser_click` | Click element by CSS selector |
| `browser_type` | Type text into input field |
| `browser_screenshot` | Take screenshot + optional visual analysis |
| `browser_close` | Close browser |

## Example: Login Automation

```typescript
// AI-generated tool calls for login flow
browser_open({ headless: false })
browser_navigate({ url: "https://app.example.com/login" })
browser_type({ selector: "input[name='email']", text: "user@example.com" })
browser_type({ selector: "input[name='password']", text: "secret123" })
browser_click({ selector: "button[type='submit']" })
browser_screenshot({ prompt: "Verify login succeeded" })
```

## Privacy & Security

| Aspect | Detail |
|:-------|:-------|
| **Local only** | Browser runs on your machine, no remote control |
| **No tracking** | Puppeteer doesn't send data externally |
| **Permission** | Requires explicit `allow` for browser operations |
| **Sandbox** | Runs in isolated process, cleaned up after use |
| **Headless default** | Runs without GUI unless you specify `headless: false` |

## Configuration

Enable browser automation in `.helixmind/config.json`:

```json
{
  "permissions": {
    "browserAutomation": true
  },
  "browser": {
    "headless": true,
    "timeout": 30000,
    "userAgent": "HelixMind/1.0"
  }
}
```
