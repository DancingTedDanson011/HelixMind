# CLI Reference

## Main Commands

### `helixmind` / `helixmind chat`

Start an interactive AI coding session (default command).

| Flag | Description |
|------|-------------|
| `-m, --message <text>` | Send a single message instead of interactive mode |
| `--yolo` | Auto-approve ALL operations (including dangerous) |
| `-s, --skip-permissions` | Skip safe/ask prompts (still asks for dangerous) |
| `--no-validation` | Disable Validation Matrix |
| `--validation-verbose` | Show detailed validation results |
| `--validation-strict` | Treat warnings as errors |

```bash
helixmind                                # Interactive chat
helixmind chat -m "Add error handling"   # Single message
helixmind --yolo --no-validation         # Full auto, no checks
helixmind -s --validation-strict         # Skip permissions, strict validation
```

### `helixmind helix`

Alias for `helixmind chat`.

### `helixmind login`

Authenticate with the HelixMind web platform.

| Flag | Description |
|------|-------------|
| `--api-key <key>` | Use API key directly (skip browser flow) |
| `--url <url>` | Custom web platform URL |
| `--force` | Re-authenticate even if already logged in |

```bash
helixmind login                          # Browser-based login
helixmind login --api-key hm_abc123      # Direct key entry
helixmind login --force                  # Re-authenticate
```

### `helixmind logout`

Remove stored authentication.

| Flag | Description |
|------|-------------|
| `--no-revoke` | Do not revoke the API key on the server |

```bash
helixmind logout                # Logout + revoke key
helixmind logout --no-revoke    # Local logout only
```

### `helixmind whoami`

Show current authentication status (email, plan, masked API key).

```bash
helixmind whoami
```

### `helixmind init`

Initialize HelixMind in the current project:

```bash
cd your-project && helixmind init
```

Creates: `.helixmind/` directory with `context.md` + `spiral.db`.

### `helixmind config`

```bash
helixmind config set <key> <value>    # Set a value
helixmind config get <key>            # Get a value
helixmind config list                 # List all settings
```

| Key | Description | Default |
|-----|-------------|---------|
| `provider` | LLM provider | `anthropic` |
| `apiKey` | API key (masked) | â€” |
| `model` | Model name | Provider default |
| `logLevel` | Debug level | `info` |
| `dataDir` | Data directory | `~/.helixmind/` |
| `maxTokens` | Max context window | Provider default |
| `validationEnabled` | Validation Matrix | `true` |
| `brainScope` | Brain scope | `local` |

### `helixmind spiral`

```bash
helixmind spiral status    # Nodes, levels, storage
helixmind spiral search    # Semantic search
helixmind spiral compact   # Trigger compaction
```

### `helixmind feed`

```bash
helixmind feed README.md src/     # Specific files/dirs
helixmind feed --deep src/        # Deep analysis
helixmind feed --watch src/       # Watch for changes
```

### `helixmind export` / `helixmind import`

```bash
helixmind export ./backups                # Export to ZIP
helixmind export -n my-project            # With name
helixmind import backup.zip              # Merge import
helixmind import --replace backup.zip    # Replace import
```

## Exit Codes

| Code | Meaning |
|------|---------|
| `0` | Success |
| `1` | General error |
| `130` | Interrupted (Ctrl+C) |
